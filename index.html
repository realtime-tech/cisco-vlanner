Basic VLAN Commands

show vlan brief - View all VLANs

show vlan id [number] - View specific VLAN

show interfaces trunk - View trunk ports

show interfaces switchport - View port details

VLAN Types

Data VLAN: Regular user traffic (1-4094)

Voice VLAN: VoIP phones

Management VLAN: Switch management

Native VLAN: Untagged traffic on trunk

Trunk Encapsulation

802.1Q: Industry standard

ISL: Cisco proprietary (legacy)

Default: 802.1Q on modern switches

Port Modes

switchport mode access - Single VLAN

switchport mode trunk - Multiple VLANs

switchport mode dynamic auto - DTP negotiation

VLAN Ranges

Normal: 1-1005 (stored in vlan.dat)

Extended: 1006-4094 (VTP transparent)

Reserved: 0, 1, 4095

Inter-VLAN Routing Methods

Router-on-a-Stick: One physical interface

Layer 3 Switch: SVIs for each VLAN

Routed Port: no switchport command
